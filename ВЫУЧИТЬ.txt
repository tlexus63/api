ПУБЛИКАЦИЯ ФАЙЛОВ 
Git clone – клонирует репозиторий
Git init – создает/ инициализирует новый репозиторий (в текущей папке репоз созд)
Git add – добавляет файл 
Git commit -m “комментарий” – запись изменения в репозиторий (внесение комита)
Git remote add origin ссылка на репозиторий – создание записи о новом подключении к удаленному репозиторию
Git push – выгрузка содержимого локального репозитория в удаленный
